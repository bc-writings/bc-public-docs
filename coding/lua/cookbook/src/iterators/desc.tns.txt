====================
Des boucles "maison"
====================

Nous proposons deux \techs pour créer ses propres générateurs — ou itérateurs, pour reprendre la terminologie courante. L’exemple présenté, bien que très simple, illustre clairement le principe et laisse entrevoir des cas plus avancés sans difficulté. Notre objectif est de pouvoir employer le code \lua suivant afin d’obtenir la sortie indiquée ci-après.


lua::
    ---
    title = Code saisi
    ---

    for x, y, z in xyz_range(2, 3, 4) do
      print(x, y, z)
    end


lua::
    ---
    title = Sortie souhaitée
    file  = ./coroutine/main.lua
    output = yes
    code   = no
    ---


content::
    ./closure/\desc
    ./coroutine/\desc
