Installation sous Latex des polices pour les calculatrices TI-82 Stats FR, Casio Graph 35+

Par Rémi Grolleau le jeudi, octobre 28 2010, 14:36

  • calculatrices
  • enseignement
  • latex
  • linux
  • mathématiques

Présentation

La réalisation de travaux pratiques utilisant la calculatrice est toujours un exercice délicat. Aux difficultés techniques s'ajoutent toujours d'inombrables difficultés d'ordre pratique qui peuvent transformer une séance de TP en véritable parcours du combattant... L'utilisation de polices sous Latex permettant d'insérer directement les touches de la calculatrices dans les documents simplifie un peu le problème.

Ce billet détaille l'installation des polices sous GNU/Linux avec la distribution Tex Live pour les calculatrices TI-82 Stats FR et Casio Graph 35+. L'ensemble des fichiers fournis a été réalisé en suivant l'excellent tutorial disponible sur cette page : Fontes casio et texas. Vous y trouverez des explications pas à pas pour créer les polices à partir des fichiers mis en lignes par les constructeurs, ainsi que différentes archives contenant l'ensemble des fichiers à installer pour profiter directement du travail réalisé par différents enseignants.

J'ai complété ces travaux pour mes besoins personnels en ajoutant quelques symboles pour la TI-82 Stats FR, ainsi que l'ensemble des touches pour la Casio Graph 35+. Afin d'harmoniser au maximum les noms des commandes latex pour les deux modèles, une grande partie des commandes diffère mais rien ne vous empêche de les changer en éditant les fichiers de style associés. Ci-dessous les fichiers regroupant l'ensemble des commandes définies pour vous donner une idée :
- ti82statsindex.pdf (.tex)
- casiographindex.pdf (.tex)

Installation

  • Télécharger l'archive Latex-PolicesCalculatrices.tar.gz. Vous y trouverez :

- un document texte expliquant l'installation des polices
- les polices d'origine téléchargées sur les sites de Texas Instrument et Casio
- les fichiers créés avec le logiciel Fontforge
- les tables des symboles
- les fichiers de styles
- les index des commandes définies

  • Décompresser l'archive dans votre dossier personnel (/home/nomdutilisateur/)
  • Ouvrir un terminal et placer vous dans le répertoire calculatrices en tapant la commande :

cd ~/calculatrices/

  • Créer les répertoires suivants :

mkdir ~/.texmf-var/fonts/afm/ti/
mkdir ~/.texmf-var/fonts/tfm/ti/
mkdir ~/.texmf-var/fonts/type1/ti/
mkdir ~/.texmf-var/fonts/enc/ti/
mkdir ~/.texmf-var/fonts/afm/casio/
mkdir ~/.texmf-var/fonts/tfm/casio/
mkdir ~/.texmf-var/fonts/type1/casio/
mkdir ~/.texmf-var/fonts/enc/casio/

  • Copier les fichiers *.tfm, *.afm, *.pfb, *.enc et *.map dans les répertoires adéquats :

cp ti82statsfr/ti82statsfr.afm ~/.texmf-var/fonts/afm/ti/
cp ti82statsfr/ti82statsfr.tfm ~/.texmf-var/fonts/tfm/ti/
cp ti82statsfr/ti82statsfr.pfb ~/.texmf-var/fonts/type1/ti/
cp ti82statsfr/ti82statsfr.enc ~/.texmf-var/fonts/enc/ti/
cp ti82statsfr/ti.map ~/.texmf-var/fonts/map/dvips/updmap/ti.map
cp casiograph/casiograph.afm ~/.texmf-var/fonts/afm/casio/
cp casiograph/casiograph.tfm ~/.texmf-var/fonts/tfm/casio/
cp casiograph/casiograph.pfb ~/.texmf-var/fonts/type1/casio/
cp casiograph/casiograph.enc ~/.texmf-var/fonts/enc/casio/
cp casiograph/casio.map ~/.texmf-var/fonts/map/dvips/updmap/casio.map

  • Ajouter les lignes Map ti.map et Map casio.map au fichier ~/.texmf-config/updmap.d/10local.cfg

echo "Map ti.map" >> ~/.texmf-config/updmap.d/10local.cfg
echo "Map casio.map" >> ~/.texmf-config/updmap.d/10local.cfg

  • Créer les répertoires ~/.texmf-var/tex/latex/ti82statsfr/ et ~/.texmf-var/tex/latex/casiograph/ :

mkdir ~/.texmf-var/tex/latex/ti82statsfr/
mkdir ~/.texmf-var/tex/latex/casiograph/

  • Copier les fichiers de style ti82statsfrfont.sty et casiograph.sty ainsi que uti82stats.fd et ucasiograph.fd :

cp ti82statsfr/ti82statsfrfont.sty ~/.texmf-var/tex/latex/ti82statsfr/
cp ti82statsfr/uti82statsfr.fd ~/.texmf-var/tex/latex/ti82statsfr/
cp casiograph/casiographfont.sty ~/.texmf-var/tex/latex/casiograph/
cp casiograph/ucasiograph.fd ~/.texmf-var/tex/latex/casiograph/

  • Enfin taper les commandes suivantes :

sudo updmap
sudo update-updmap
sudo mktexlsr
Je ne connais pas le rôle de ces commandes, ni s'il est nécessaire de les lancer en tant que superutilisateur (précédées de sudo)... En cas de problème, les retaper plusieurs fois.

Utilisation

Pour utiliser les polices des calculatrices dans un document Latex, il suffit désormais juste d'ajouter dans le préambule :
\usepackage{ti82statsfrfont} % touches TI-82 Stats FR
\usepackage{casiographfont} % touches Casio Graph 35+
Il ne reste plus qu'à taper la commande \timode pour obtenir la touche "MODE" d'une TI-82 Stats FR par exemple...

Essayer de compiler les fichiers ti82statsindex.tex et casiographindex.tex pour tester votre installation.

Pour plus de détails sur la façon dont les polices ont été crées et vérifier l'installation des polices, je vous engage à lire cette page sur laquelle se base cette documentation.
