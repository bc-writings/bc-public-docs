README.md

tikzdice

A LaTeX/Tikz function library to draw six-sided dice icons in a few colors.

I'm teaching a math/stats/decision-science class and I needed to make some examples using dice. I found a few posts on stackexchange (here and here) and it seemed like a straightforward thing to build a simple function library to accommodate my needs. Seemed like a good idea to share.
