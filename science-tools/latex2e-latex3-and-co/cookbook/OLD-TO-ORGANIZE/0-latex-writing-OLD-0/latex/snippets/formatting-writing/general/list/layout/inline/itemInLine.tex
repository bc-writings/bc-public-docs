On considère un parallélogramme $ABCD$.

Démontrer que :

\begin{inparaenum}
	\item $\vv{BA}+\vv{DA}=\vv{CA}$;\hfill
	\item $\vv{AD}+\vv{CB}=\vv{0}$;\hfill
	\item $\vv{DC}+\vv{BC}=\vv{AC}$.
\end{inparaenum}

Démontrer que :

\begin{multicols}{3}
	\begin{enumerate}
		\item $\vv{BA}+\vv{DA}=\vv{CA}$;
		\item $\vv{AD}+\vv{CB}=\vv{0}$;
		\item $\vv{DC}+\vv{BC}=\vv{AC}$.
	\end{enumerate}
\end{multicols}
