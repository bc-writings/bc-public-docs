            Dans une distribution LaTeX (ou TeX), le mécanisme de recherche des fichiers est assuré par la bibliothèque kpathsea associée à une base de donnée de chemins de recherche décrite dans un des fichiers de configuration texmf.cnf..
            La commande

            Code : Tout sélectionner

            texdoc kpathsea
            le décrit complètement.
            La commande kpsewhich permet de l'activer de manière autonome.
            On peut l'utiliser pour vérifier qu'un fichier inclus dans un source LaTeX sera trouvé, y compris les fichiers images et les fichiers sty.
            Par exemple, sous Linux, (et peut être sous MacOS) la commande

            Code : Tout sélectionner

            less $(kpsewhich amsmath.sty)
            permet de parcourir le fichier amsmath.sty.
            JJR.
