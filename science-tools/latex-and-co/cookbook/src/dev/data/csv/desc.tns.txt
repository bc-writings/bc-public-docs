==============================
Usage basique de fichiers \csv
==============================

Considérons le fichier \csv suivant.

csv::
    ---
    title = Les données brutes.
    path  = ./data.csv
    ---


Nous allons voir comment utiliser les données précédentes pour produire le résultat suivant.

pdf::
    ---
    title = Rendu souhaité.
    path  = ./main.pdf
    ---


note::
    Nous avons choisi de lire complètement le fichier \csv pour construire un stockage en interne des données, celles-ci étant rendues accessibles à l'utilisateur final via des macros simples à employer.
