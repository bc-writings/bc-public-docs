==============================
Usage basique de fichiers \csv
==============================

Considérons le fichier \csv suivant.

csv::
    ---
    title = Les données brutes.
    path  = ./data.csv
    ---


Nous allons voir comment utiliser le fichier précédent afin de produire le résultat ci-après.

pdf::
    ---
    title = Rendu souhaité.
    path  = ./main.pdf
    ---


note::
    L'idée de cette recette est de tout faire à la main sans faire appel à un \pack dédié.
    Le choix \tech retenu est de lire complètement le fichier \csv pour construire un stockage en interne des données, puis de rendre les données accessibles à l'utilisateur final via des macros simples à employer.
