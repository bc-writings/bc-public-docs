==============
Version animée
==============

Le code proposé ici reprend la recette précédente en faisant tourner le cube, puis en produisant un \pdf dynamique via le package ''animate''.


caution::
    Tous les lecteurs \pdf ne supportent pas les animations créées avec ''animate''. Vous pouvez utiliser le lecteur intégré au navigateur \firefox.
